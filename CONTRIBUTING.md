## Getting Started

1. **Fork** the repository
2. **Clone** your fork:

   - HTTPS

      ```bash
      git clone https://github.com/imShakil/100-Days-Of-DevOps-Challenge-KodeKloud.git
      ```

   - Or SSH

      ```sh
      git clone git@github.com:imShakil/100-Days-Of-DevOps-Challenge-KodeKloud.git
      ```

3. **Create** a branch:

   ```bash
   git checkout -b feature/your-improvement
   ```

4. **Make** your changes
5. **Test** your solutions
6. **Commit** with clear messages:

   ```bash
   git commit -m "Fix: Correct Kubernetes deployment configuration"
   ```

7. **Push** and create a Pull Request

## Guidelines

### Solution Standards

- **Test thoroughly** - Verify commands work as expected
- **Document clearly** - Include step-by-step instructions
- **Follow format** - Match existing solution structure
- **Add context** - Explain why specific approaches are used

### Code Quality

- Use consistent formatting
- Include error handling where appropriate
- Add comments for complex configurations
- Follow best practices for each technology
